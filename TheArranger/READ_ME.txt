There are some external (Jar`s) that must be added to the java to make the software work.
you can find these jar with in (TheArranger)File.


to add it inside the Java "e.g eclips":
1- right click on the project filename (TheArranger) in "Package Explorer" .
2- click on Build Path.
3- click on confgure Build Path.
4-click on library tap and then click on (Add External Jar`s).
5- hughlight all the jar`s in the project file (TheArranger).
6-press 'OK' and compile.
